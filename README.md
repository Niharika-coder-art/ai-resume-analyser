AI Resume Analyzer

📌 Project Description

AI Resume Analyzer is a web application that analyzes resumes and provides insights such as ATS score, skill analysis, and job recommendations.

🚀 Features

- Upload resume (PDF/DOCX)
- Analyze skills from resume
- ATS score prediction
- Suggest improvements
- Generate interview questions

🛠 Technologies Used

- React / Next.js
- JavaScript
- HTML & CSS
- AI APIs
