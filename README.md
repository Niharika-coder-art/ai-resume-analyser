📌 Project Overview

AI Resume Analyzer is a web application that analyzes resumes and provides intelligent insights to help job seekers improve their chances of getting selected by recruiters. The system evaluates resumes, identifies important skills, and suggests improvements to make the resume more effective and ATS-friendly.

This project demonstrates the use of Artificial Intelligence concepts to process and analyze resume content.

---

🚀 Features

- 📄 Upload Resume (PDF or Document)
- 🔍 Analyze resume content
- 📊 ATS Score Prediction
- 🧠 Skill extraction from resume
- 💡 Suggestions to improve resume quality
- 🎯 Generate possible interview questions based on skills
- 📑 Display structured analysis results

---

🛠️ Technologies Used

- Frontend: React / Next.js
- Programming Language: JavaScript
- Styling: CSS / Tailwind CSS
- AI Integration: AI-based text analysis
- Tools: VS Code, GitHub

🎯 Purpose of the Project

The main goal of this project is to help job seekers understand how their resumes are evaluated and improve them before applying for jobs. It also demonstrates practical implementation of AI concepts in a real-world application.

💻 How to Run the Project

1. Clone the repository
2. Open the project in VS Code
3. Install dependencies

npm install

4. Run the project

npm run dev
